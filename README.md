# Abdul Omope — Portfolio Website

A personal portfolio website for Abdul Omope, a Marketing & Communications Strategist based in Calgary, AB.

## About

A single-page portfolio showcasing a decade of marketing and communications experience across Canada and Nigeria. The site includes:

- **Hero Section** — Introduction with photo, key stats (10+ years, 1B+ impressions, 100+ brands), and professional summary
- **Quick Background** — Snapshot of current role, previous employers, location, target roles, education, and languages
- **About Me** — Personal story from building a first website at 15 to leading campaigns across two continents
- **Featured Writing** — Curated articles on marketing, career, culture, place, and the immigrant experience
- **Real Results** — Metrics and outcomes from past roles with expandable "View All" toggle
- **Experience** — Full work history timeline (9 roles) with expandable "View All" toggle
- **Core Competencies** — 8 skill areas with expandable "View All" toggle
- **Brands** — Organizations worked with across Canada and Nigeria
- **Testimonials** — Recommendations from colleagues and managers
- **Tech Stack** — Tools and platforms (Drupal, WordPress, Google Analytics, Mailchimp, etc.)
- **FAQs** — Accordion-style dropdown answering common hiring questions (work authorization, role history, flexibility)
- **Contact** — Email button and social media icons (LinkedIn, X, Instagram, Facebook)

## Tech

Single-page static site built with HTML, CSS, and vanilla JavaScript. No frameworks or build tools required.

## Deployment

This site can be deployed for free using **GitHub Pages**:

1. Create a new repository on GitHub
2. Upload all files from this folder
3. Go to repo **Settings** > **Pages**
4. Under "Source", select **Deploy from a branch**
5. Choose `main` branch and `/ (root)` folder
6. Click **Save**
7. Your site will be live at `https://yourusername.github.io/portfolio-website/`

### Direct Section Links

Once hosted, you can link directly to any section:
- `yourdomain.com/#about` — About Me
- `yourdomain.com/#experience` — Experience
- `yourdomain.com/#faq` — FAQs (useful for cover letters)
- `yourdomain.com/#contact` — Contact
