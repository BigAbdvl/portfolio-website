## Tech

Single-page static site built with HTML and CSS. No frameworks or build tools required.

## Deployment

This site can be deployed for free using **GitHub Pages**:

1. Go to your repo **Settings** > **Pages**
2. Under "Source", select **Deploy from a branch**
3. Choose `main` branch and `/ (root)` folder
4. Click **Save**
5. Your site will be live at `https://yourusername.github.io/portfolio-website/`
